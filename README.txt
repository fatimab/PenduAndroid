# PenduAndroid
 
Ce projet est un jeu du pendu JAVA sous android : 
Des mots sont générés aléatoirement à partir d’un fichier txt 
Le but est de trouver le mot en 6 essais
Les lettres essayées sont écrites au fur et à mesure à l’écran, et un message apparaît quand on rentre une lettre qu’on a déjà testée (mais l’erreur est comptabilisée quand même).
Quand on gagne, une popup s’ouvre pour dire qu’on a gagné et un bouton permet de rejouer.
Quand on perd, un message s’affiche de la même manière, et la solution s’affiche en plus.
